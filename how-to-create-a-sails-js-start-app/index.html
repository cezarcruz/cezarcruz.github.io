  • Hello.js (api/models/)
    • A common empty model file.

To create a new endpoint, you need to edit the HelloController.js and add the code bellow:

module.exports = {
        action: function (req, res) {
                return res.send('Hello!');
        }
};

In this code, “action” is the name of the endpoint. The endpoint will be composed by: Controller name (without ‘Controller’) and the action name, in our case, will be: http://localhost:1337/hello/action

Now you can start Sails again, but this time, it will ask you what to do with Model. To this example, you just need answer “2”, then, your app will start normally.

Now, go to http://localhost:1337/hello/action , and you will see the “Hello!” message.
