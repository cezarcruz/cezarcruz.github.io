Olas, Para habilitar esta funcionalidade no nautilus do Fedora eh muito simples, basta rodar os comandos abaixo: sudo dnf install nautilus-open-terminal Apos isto, basta reiniciar o nautilius, e quando clicar com o botão direito do mouse, a opção “Open in terminal” estará disponível: killall nautilus Prontinho! =)
